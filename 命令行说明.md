#### 编译

首先在程序文件夹下编译所有的程序：

```
javac -encoding UTF-8 *.java
```

#### 设置环境变量

在系统变量中添加 变量名：CLASSPATH，变量值：.class文件所在的路径

这样能保证.class在任意目录下都能运行

#### 程序运行

然后切换到需要建立仓库的文件夹，运行下面的语句建立仓库（初始化）

```
java gitCommand init
```
